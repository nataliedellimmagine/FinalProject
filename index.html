HTML5:

HTML5 is the code used to create the structure of a website or app. It codes the placement of the text and images. HTML stands for Hypertext Markup Language.


CSS3:

CSS3 is used to style a webpage or app. It is used to change the colors and spacing and make it look pretty. CSS stands for Cascading Style Sheets.


JavaScript:

JavaScript or JS is used for user interactions. For example, JS is used to code a button that changes a tab or makes text appear and disappear.


Frontend and Backend:

Frontend programming is the User Interface and how consumers interact with a webpage or app. Backend programming is the data that makes up the webpage or app.
